Progress audio assets
=====================

`progress-keyboard.ogg` is a bundled local progress loop for backend-opt-in first-token wait feedback.

Source: Wikimedia Commons, `File:Computer keyboard.ogg`
https://commons.wikimedia.org/wiki/File:Computer_keyboard.ogg

Author/source notes from Commons: russiandoll / PDSounds, "Fast copy-typing on chunky-keyed wireless PC keyboard. London, May 2008."

License: public domain. The Commons file page states that the author released the work into the public domain and grants use for any purpose where public-domain release is not legally possible.

Runtime rule: Mortic must serve this local file from the app bundle. Do not fetch external audio during an active voice turn.
